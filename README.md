# mini-isomorphic-react

The world's smallest isomorphic React demo.


## Install

```bash
$ git clone git@github.com:jedireza/mini-isomorphic-react.git && cd ./mini-isomorphic-react
$ npm install
```


## Start the app

```bash
$ npm start
```
